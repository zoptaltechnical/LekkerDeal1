Convert any kind of HTML Content into PDF using shortcode.
 
== Description ==

HTML PDF Generator is a best way to generate a PDF for any kind of HTML Content using shortcode in page/post editor.


**More for info & Demo**

<a href="https://desirepress.com/plugin/html-pdf-generator/" target="_blank">Click Here</a>

= Features =

* Convert Images into PDF.
* Convert HTML Content into PDF.
* Convert HTML Table into PDF.
* Convert Plain Text into PDF.
* Can use Multiple Shortcodes on a single page.

= Usage =

It is really simple, just use PDF shortcode (like below) in page/post editor.

[pdfg_shortcode]This HTML/Content will be convert into PDF format.[/pdfg_shortcode]

== Installation ==

* Upload the plugin folder to the /wp-content/plugins/ folder of your WordPress installation.  
* Activate the plugin.

Enjoy it......
